## 完整数据流

```
方案选择(HomeView)
  → 查看详情(RecipeDetail)
  → 准备确认(BrewingView)
  → 开始计时(BrewingBottomSheet)
  → 完成冲煮
  → 品鉴记录(TastingNotesView)
  → 保存记录
  → 历史查看(HistoryView/RecordDetail)
  → 再次冲煮循环
```
